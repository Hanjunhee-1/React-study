★  List

    같은 아이템들을 순서대로 모아놓은 것을 List 라고 볼 수 있다.
    List 를 위해 사용하는 자료구조는 Array 이다.

    Array 는 JavaScript 의 변수나 객체들을 하나의 변수로 묶어놓은 것이다.
    아래의 예시 코드는 JavaScript 의 배열이 무엇인지를 보여준다.
    ```
        const numbers = [1,2,3,4,5];
    ```

★  Key

    Key 는 열쇠를 뜻한다. 예를 들어 이웃집 열쇠와 우리집 열쇠가 똑같은 열쇠라면
    해당 열쇠는 열쇠로서의 기능을 하지 못하게 된다. 때문에 Key 는 각 객체나 아이템을 구분해주는
    고유한 값이라고 볼 수 있다.
    List 에서의 Key 는 아이템들을 구분하기 위한 고유한 문자열이다.
