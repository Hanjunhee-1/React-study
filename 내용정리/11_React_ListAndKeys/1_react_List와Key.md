# 🔔 List

- List 는 배열과 같습니다. 어떤 요소들을 순서대로 모아놓은 것을 List 라고 합니다.

- 아래는 List 의 예시입니다.
    ```js
        const numbers = [1,2,3,4,5];

        const fruits = ['apple', 'banana', 'coconut'];

        const users = [
            {
                id: 1,
                name: 'kevin',
                age: 21
            },
            {
                id: 2,
                name: 'tom',
                age: 30
            },
            {
                id: 3,
                name: 'alice',
                age: 25
            }
        ];
    ```
    <br/><br/><br/>

# 🔔 Key

- Key 는 열쇠를 뜻합니다. Key 의 특징으로는 고유성을 갖는다는 것인데 이러한 특징 덕분에 List 내부에서 어떠한 요소를 다른 객체나 아이템과 구분해줄 수 있습니다.
