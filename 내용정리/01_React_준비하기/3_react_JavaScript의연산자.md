# 🔔 JavaScript 의 연산자
- 기본 사칙연산이나 비교연산과 같은 것들은 이미 알고 있다 생각하고 넘어가도록 하겠습니다

- `일치연산자`
    - `JavaScript` 에는 `===`, `!==` 과 같은 `일치연산자` 라는 것이 존재합니다.
    - `===` 연산자는 비교하려는 변수들끼리 값과 자료형이 모두 같다면 `true` 를 반환합니다.
    - `!==` 연산자는 비교하려는 변수들끼리 값 또는 자료형이 하나라도 다르다면 `true` 를 반환합니다.
    - 예시
        ```Javascript
            let a = 1;      // number
            let b = '1';    // string

            a == b          // true
            a === b         // false
        ```
