# ❓ JavaScript 란
- 만약 웹 사이트가 HTML 과 CSS 로만 이루어져있따면 버튼을 누르고 그에 대한 반응을 하는 등의 동적인 작업을 할 수가 없습니다. 때문에 동적인 작업의 역할을 해줄 언어가 필요한데 그것이 바로 `JavaScript` 입니다.

- `JavaScript` 는 프로그래밍 언어의 한 종류로 이름에 `Java` 가 들어가서 `Java` 와 연관되었다고 생각할 수도 있지만 `JavaScript` 의 정식 명칭은 `ECMAScript` 입니다. 

- 버전을 매길 때 `ES5`, `ES6` 등으로 이름을 짓습니다.

- `JavaScript` 와 같은 `Script Language` 의 특징은 프로그램이 실행되는 `Run Time` 에 코드가 해석되고 실행된다는 것입니다. <br/><br/><br/>


# 🔔 JavaScript 의 자료형들 (간단한 예시)
```Javascript
let a = 1;          // number

let b = "hello";    // string

let c = true;       // boolean

let d = null;       // null

let e;              // undefined

let f = [1,2,3];    // array

let g = {
    a: 'apple',
    b: 'banana',
    c: 'carrot',
};                  // object
```
