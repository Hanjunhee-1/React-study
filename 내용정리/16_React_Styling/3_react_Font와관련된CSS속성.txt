★  font-family

    글꼴과 관련해서 스타일을 설정할 때 사용한다.
    예시는 다음과 같다.
    ```
        #title {
            font-family: "Times New Roma", Times, serif;
        }
    ```
    글꼴은 사용할 글꼴을 맨 처음에 쓰고 대비책으로 여러가지를 써놓을 수 있다.
    참고로 사용할 글꼴에 띄어쓰기가 포함된다면 따옴표로 묶어주어야 한다.

    가장 마지막에 있는 것은 글꼴의 형태를 구분하는 것인데 일반적인 분류는 다음과 같다.

    (1) serif
        각 글자의 모서리에 작은 테두리를 갖고 있는 형태의 글꼴이다.

    (2) sans-serif
        모서리에 테두리가 없이 깔끔한 선을 가진 글꼴이다.
        컴퓨터 모니터에서는 serif 보다 가독성이 좋다.

    (3) monospace
        모든 글자가 같은 가로 길이를 가지는 글꼴이다.
        코딩을 할 때 주로 사용한다.

    (4) cursive
        사람이 쓴 손글씨 모양의 글꼴이다.

    (5) fantasy
        장식이 들어간 형태의 글꼴이다.


★  font-size

    글자의 크기와 관련해서 스타일을 설정할 때 사용한다.

    단위로는 대표적으로 다음과 같다.    
    
    (1) px
    (2) em
    (3) rem
    (4) vw
    (5) vh

    1em = 16px 
    라는 것을 기억해두자.


★  font-weight

    글자의 굵기와 관련해서 스타일을 설정할 때 사용한다.


★  font-style

    이름 그대로 글꼴의 스타일을 설정하기 위해 사용한다.
    일반적으로 다음과 같은 값들이 있다.

    (1) normal
        일반적인 글자의 형태를 의미한다.

    (2) italic
        글자가 기울어진 형태로 나타난다.
        별도로 기울어진 형태의 글자들을 직접 디자인해서 만든 것이다.

    (3) oblique
        글자가 비스듬한 형태로 나타난다.
        그냥 글자를 기울인 것이다.
