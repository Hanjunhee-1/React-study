★  background-color

    배경 색을 바꾸기 위해 사용한다.

    색상값은 다음과 같은 것들이 있다.

    (1) 16진수 컬러값
        색상을 16진수로 표현한다. 예를 들면 #ff0000 이런 식으로 말이다.

    (2) 투명도를 가진 16진수 컬러값
        예를 들면 #ff000055 이런 식으로 말이다.

    (3) rgb 컬러값
        16진수로 쓰던 것을 좀더 보기 쉽게 우리가 아는 10진수로 바꿔쓴 것이다.
        예를 들면 rgb(255, 0, 0) 이런 식으로 말이다. 

    (4) rgba 컬러값
        rgb 컬러값과 동일한데 투명도를 추가해준다.
        rgba(255, 0, 0, 0.5) 이런 식으로 말이다.

    (5) 미리 정의된 색상의 이름
        그냥 미리 정의된 색상을 사용할 수도 있다. 
        예를 들면 red, blue, green, yellow, black, white 등이 있다.

    
★  border

    테두리 선의 스타일을 설정할 때 사용한다.
    예를 들면 다음과 같이 사용한다.

    ```
        div {
            border: 1px solid black;
        }
    ```
    선의 굵기가 1px 이고 직선(solid) 이고 색은 검정색인 테두리 선을 만들어준다.
