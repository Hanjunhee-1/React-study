# 🔔 CSS 

- CSS 는 Cascading Style Sheets 의 약자로 웹 페이지를 더욱 아름답게 꾸며주기 위한 언어입니다.

- 선택자는 Selector 라고 합니다. 
    ```css
        h1 {
            color: green;
            font-size
        }
    ```
    - 위의 css 예시에서 h1 은 html 태그이면서 선택자입니다. html 태그 외에도 지정해준 class 이름 혹은 id 값이 올수도 있습니다. 

    - class 의 경우에는 dot(.) 으로 접근하고 id 의 경우에는 sharp(#) 으로 접근합니다.
