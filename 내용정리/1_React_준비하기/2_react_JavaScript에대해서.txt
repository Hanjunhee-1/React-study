★  JavaScript 란?
    만약 웹 사이트가 HTML 과 CSS 로만 이루어져있다면 버튼을 누르는 등의 동적인 작업을 할 수 없다.
    때문에 동적인 작업의 역할을 해줄 언어가 필요한데 그것이 바로 JavaScript 이다.

    JavaScript 는 프로그래밍 언어의 한 종류이다. 이름에 Java 가 들어가서 Java 와 연관된 프로그래밍 언어라고 생각할 수 있지만
    JavaScript 의 정식 명칭은 ECMAScript 이다. 버전을 매길 때 ES5, ES6 등으로 이름을 짓는다.

    JavaScript 같은 Script Language 의 특징은 프로그램이 실행되는 Run time 에 코드가 해석되고 실행된다는 것이다.
    JavaScript 의 사용범위는 굉장히 넓다.


★  JavaScript 의 자료형

    number, string, boolean, null, undefined, array, object 타입이 있다.

    let a = 1; --> number

    let b = "hello"; --> string

    let c = true; --> boolean

    let d = null; --> null

    let e; --> undefined(변수를 선언만 하고 값을 대입하지 않은 것)

    lef f = [1, 2, 3]; --> array

    let g = {a: "apple", b: "banana", c: "carrot"}; --> object
