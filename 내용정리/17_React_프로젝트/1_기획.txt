그동안 배운 것들을 모두 활용해서 미니 블로그를 만들어볼 것이다.

서버가 따로 개발되어 있지 않아서 실제 블로그처럼 동작할수는 없지만
웹 페이지는 개발할 수 있다.

미니 블로그에 필요한 기능은 다음과 같다.

    1.  글 목록 보기 기능 (리스트 형태)
    2.  글 보기 기능
    3.  댓글 보기 기능
    4.  글 작성 기능
    5.  댓글 작성 기능

필요한 패키지는 다음과 같다.

    1.  react-router-dom
        페이지 전환을 위해 react 프로젝트에서는 필수적으로 사용해야 하는 패키지이다.
    
    2.  styled-components
        컴포넌트를 꾸미기 위해 사용해야 하는 패키지이다.



※  원래는 'npx create-react-app [프로젝트 이름]' 을 통해 따로 react-app 을 생성해주어야하지만
    기존에 실습을 진행하던 폴더인 'react-study' 폴더에서 진행할 것이다.
    '17_프로젝트' 라는 이름의 폴더에 기능별로 생성한 컴포넌트를 모아둘 것이다.
